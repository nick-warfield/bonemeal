Geon FX Asset Help

What is Geon FX?

Geon FX is the newest and the most advanced particle editor for GameMaker: Studio and Game Maker Studio 2.

It allows you to create complex particle effects on Windows or Mac and play them in your project easily.

Geon FX comes with plenty of effects for you to test and examine.

Where is Geon FX?

There are two archives in Included Files - editor folder. One is for macOS, another for Windows.

Extract archives and run .exe (on Windows) or .app (on macOS).

Press F1 on keyboard or click ? icon at the top-right corner of the window to show editor Help.

How to play Geon FX effects?

You can either export your effect to GML in Geon FX editor then copy and paste resulting code into your project. Or use our custom loading scripts.

We have designed a very simple system for you to use Geon FX effects. Thatâ€™s how it works:

  1. Import obj_geonfx_con object to your project;
  2. Import all scripts from geon_fx folder;
  3. Import all sprites from geon_fx folder;
  4. Add .geon effects as Included Files;
  5. Load them using geon_effect_add(filename, -1) function (check our examples);
  6. Play any effect once with geon_effect_burst(effect_ind, x, y) or continuously with geon_effect_stream(effect_ind, x, y, enable).

Examples

There are several rooms in the project. You can change their order and run whichever you want first.

  1. rm_geon_test_stream contains obj_effect_stream which streams several Geon FX effects. You can cycle through them with Left Click.
  2. rm_geon_test_burst and obj_effect_burst show how to load and play Geon FX effect once. Left Click to create an explosion.
  3. rm_geon_test_concurrent_effects shows how to play a dozen Geon FX effects simultaneously. Left Click to create explosions. 1-8 to switch fire streams on/off.
  4. obj_effect_gml_export_rain and obj_effect_gml_export_fire are two objects in corresponding rooms. They are illustrating GML export function from Geon FX editor.
